# Хата-мазанка: Оберіг села

Стратегічна браузерна гра українською мовою про відновлення традиційної хати-мазанки. Гра працює без збірки, серверної частини або зовнішніх залежностей — достатньо відкрити `index.html` у браузері.

## Основні можливості

- **3 рівні складності:** мала мазанка, велика родинна хата та музейна окраса села.
- **Візуальна еволюція хати:** підмурок, стіни, стріха й оздоблені вікна зʼявляються поступово зі зростанням «серця хати».
- **Сусіди-NPC:** Пан Василь, Маруся та Дід Остап дають різні бонуси й короткі поради в діалоговій бульбашці.
- **Щоденні події:** дощ, ярмарок, вечірня пісня або мандрівний різьбяр змінюють перебіг дня.
- **Менеджмент ресурсів:** запаси, громада, рідкісні оздоби та втома впливають на кожне рішення.
- **Збереження прогресу:** поточний рівень і розблокування автоматично зберігаються в `localStorage`.
- **Звукові заготовки:** кліки й переможна народна мелодія відтворюються через Web Audio API, якщо браузер це підтримує.

## Як грати

1. Відкрийте `index.html` у сучасному браузері.
2. Обирайте дії внизу екрана, стежачи за запасами та втомою.
3. Запрошуйте сусідів, щоб швидко підняти громаду й отримати сюжетні підказки.
4. Якщо втома сягає 100%, потрібно витратити хід на відпочинок.
5. Виконайте вимоги рівня до завершення доступних днів, щоб розблокувати наступну хату.
