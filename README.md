# Хата-мазанка: Оберіг села

Професійна браузерна стратегія українською мовою про відбудову традиційної хати-мазанки за сім ігрових днів.

## Особливості

- **Пʼять рівнів складності:** від молодої родинної хати до легендарної святкової хати для ярмарку.
- **Динамічна економіка:** глина, запаси, громада, натхнення, втома, обереги та прогрес «Серця хати».
- **Семиденний цикл:** кожна дія завершує день, а випадкові ранкові події змінюють план.
- **Соціальна система:** до двору приходять Дід Василь, Сусідка Маруся, Сільська молодь, Майстриня Оксана та Кобзар Левко.
- **Більше ідей дій:** скласти піч, влаштувати вечорниці, сплести оберіг, розмалювати причілок після 50% прогресу.
- **Візуальна еволюція:** SVG-хата проходить 5 станів — глиняні стіни, побілка, стріха, орнаменти та квіти.
- **Текстури й атмосфера:** глиняні цяточки, плетений рушник, тин, криниця, доріжка, витинанковий фон і тепла палітра.
- **Збереження:** `localStorage` памʼятає поточний рівень і найкращий результат для кожної складності.
- **Mobile first:** великі сенсорні кнопки, адаптивна сітка й вигляд автономного вебзастосунку.

## Як грати

Відкрийте `index.html` у сучасному браузері. За 7 днів потрібно довести «Серце хати» до 100%.
Плануйте дії в трьох групах:

1. **Робота:** місити стіни, крити стріху, скласти піч, розмальовувати хату після відкриття 50% прогресу.
2. **Громада:** кликати Марусю по запаси, сільську молодь для швидкої допомоги або влаштовувати вечорниці для натхнення.
3. **Відпочинок:** зменшувати втому й плести обереги, які помʼякшують негоду.

Гра не потребує збірки або зовнішніх залежностей.
