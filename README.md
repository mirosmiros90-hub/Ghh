# Хата-мазанка: Оберіг села

Невелика браузерна гра українською мовою про відновлення традиційної хати-мазанки.

## Як грати

Відкрийте `index.html` у браузері. За 7 ігрових днів потрібно набрати 100% «серця хати», використовуючи дії:

- замісити глину;
- побілити стіни;
- полагодити стріху;
- посадити мальви;
- покликати сусідів.

Гра не потребує збірки або зовнішніх залежностей.
